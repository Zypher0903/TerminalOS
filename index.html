üöÄ Getting Started

On Linux (APT-based):

sudo apt update
sudo apt install python3 python3-pyqt6 git
git clone https://github.com/Zypher0903/TerminalOS
cd TerminalOS
python3 main.py

On Windows: Download the .zip below, extract it, and run main.py with Python 3 installed.
